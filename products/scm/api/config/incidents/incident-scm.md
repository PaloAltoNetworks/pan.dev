The Unified Incidents framework consolidates incident management across multiple Palo Alto Networks security products into a single, cohesive interface within Strata Cloud Manager. This unified approach brings all incidents into one view, streamlining incident response workflows and providing consistent visibility across your entire security infrastructure.

The Unified Incidents APIs enable you to monitor and manage security and operational incidents across all Palo Alto Networks products managed by Strata Cloud Manager. Use these APIs to retrieve incident lists, access detailed incident information.

These APIs use the [common SASE authentication](/sase/docs/getstarted) for service access and authorization.
