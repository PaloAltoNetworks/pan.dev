The following table lists the API endpoint URLs for accessing CyberArk Certificate Manager - SaaS. Each endpoint URL is associated with a specific geographic region and ensures optimized network performance and compliance with regional data residency requirements. Use the endpoint that corresponds to your organization's data location or operational region when making API requests to Certificate Manager - SaaS.

| Region            | Endpoint            |
|-------------------|---------------------|
| United States     | api.venafi.cloud    |
| Australia         | api.au.venafi.cloud |
| Canada            | api.ca.venafi.cloud |
| Europe            | api.eu.venafi.cloud |
| Singapore         | api.sg.venafi.cloud |
| United Kingdom    | api.uk.venafi.cloud | 

> 📘 Note
> In each API reference page on Dev Central, a **Base URL** dropdown is available above the example code. Open the dropdown menu to select the API endpoint that matches the regional deployment your organization uses. For more information, refer to the <a href="/api-endpoints/saas/certificate-manager-saas-api" target="blank">API Endpoints Introduction</a>.
