The Data Plane supports scan execution, monitoring, and reporting operations. It provides endpoints for running scans, retrieving attack details, and generating reports.

### Key Capabilities

* **Scan Execution**: Create and manage security scan jobs (Attack Library, Agent Scan, Custom Scan).
* **Job Management**: List, monitor, and abort running scans.
* **Comprehensive Reporting**: Access detailed attack reports, multi-turn conversations, and statistics.
* **Report Downloads**: Export reports in CSV, JSON, or combined formats.
* **Custom Attack Analysis**: View custom attack results and property-based statistics.
* **Dashboard Analytics**: Get scan statistics and risk profiles.
