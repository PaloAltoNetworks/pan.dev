The Management Plane supports configuration, setup, and administrative operations. It provides endpoints for managing targets, instances, custom prompts, and templates.

### Management Plane Key Capabilities

* **Target Management**: Configure and manage AI model targets for security testing.
* **Instance and Device Management**: Set up and configure testing instances and devices.
* **Custom Attack Configuration**: Create and manage custom prompt sets and properties.
* **Template Management**: Access predefined templates and metadata for various AI providers.
* **Dashboard Overview**: Get high-level statistics and target counts.
