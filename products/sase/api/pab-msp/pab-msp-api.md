The Prisma Browser for MSPs provides the technical resources to automate security for the SMB market. This API suite allows Managed Service Providers to manage the entire lifecycle of high volume and low seat count tenants through a single interface. Developers can use these endpoints to onboard new customers with as few as one user and apply best practice security templates instantly. The guide also details how to sync multi-tenant incident data into existing SOC workflows or ServiceNow platforms using RBAC and the unified API Gateway.

These APIs use the [common SASE authentication](/sase/docs/getstarted) for service access and authorization.
