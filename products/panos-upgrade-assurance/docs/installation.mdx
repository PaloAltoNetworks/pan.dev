# Installation

The **PAN-OS Upgrade Assurance** package is not yet available in PyPI repository. The preferred method is to install it from the Git repository provided by **Palo Alto Networks**.

``` console
python -m pip install 'git+https://github.com/PaloAltoNetworks/pan-os-upgrade-assurance'
```
