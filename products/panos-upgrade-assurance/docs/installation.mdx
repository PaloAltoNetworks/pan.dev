# Installation

The **PAN-OS Upgrade Assurance** package is not yet available in the public python repository. The preferred method is to install it from the package provided by Palo Alto Networks.

``` console
pip3 install path/to/upgrade_assurance-X.Y.Z.tar.gz
```
