# Manage Custom URL Categories



Palo Alto Networks provides a set of predefined URL filtering categories.  
You can also specify your own URL filtering categories using a customer URL category object. For example, create a custom list of URLs that you want to use as match criteria in a Security policy rule. 
You can use the custom URL categories as a security policy match criteria.  

For more information, refer to the following links.
- [Where Can I Find a Complete List of PAN-DB URL Filtering Categories?](https://knowledgebase.paloaltonetworks.com/KCSArticleDetail?id=kA10g000000Cm5hCAC)
- [Palo Alto Networks URL filtering](https://urlfiltering.paloaltonetworks.com/)
- [Create a custom URL category for Cloud NGFW on AWS](https://docs.paloaltonetworks.com/cloud-ngfw/aws/cloud-ngfw-on-aws/rules-and-rulestacks/cloud-ngfw-security-rule-objects/create-a-custom-url-category-for-cloud-ngfw-on-aws)
