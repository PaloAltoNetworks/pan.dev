# Cloud NGFW Overview

Cloud NGFW is Palo Alto Networks ML-powered Next-Generation Firewall (NGFW) capability delivered as a fully managed cloud-native service by Palo Alto Networks on the Amazon Web Services (AWS) and Azure platforms.

This deployment model combines the power of the Palo Alto NGFW with the ease of use. It provides threat prevention and detection through cloud-delivered security services and threat prevention signatures.
