## Notes

<div class="note">

<div class="title">

Note

</div>

\- Panorama is supported. - Checkmode is supported. - PAN-OS
connectivity should be specified using *provider* or the classic PAN-OS
connectivity params (*ip_address*, *username*, *password*, *api_key*,
