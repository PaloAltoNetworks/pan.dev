## Synopsis

- This module works for PAN-OS 7.0 and above.
- Allows for the management of decryption rules on PAN-OS.

## Requirements

The below requirements are needed on the host that executes this module.

- pan-python \>= 0.17
- pan-os-python \>= 1.7.3
