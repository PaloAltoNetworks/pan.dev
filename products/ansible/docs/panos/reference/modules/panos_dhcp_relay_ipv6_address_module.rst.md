## Synopsis

- Manage DHCP relay IPv6 addresses on PAN-OS firewall.

## Requirements

The below requirements are needed on the host that executes this module.

- pan-python \>= 0.17
- pan-os-python \>= 1.7.3

## Notes

<div className="note">

<div className="title">

Note

</div>

\- Check mode is supported. - Panorama is not supported. - PAN-OS
connectivity should be specified using *provider* or the classic PAN-OS
connectivity params (*ip_address*, *username*, *password*, *api_key*,
and *port*). If both are present, then the classic params are ignored.

</div>

## Examples

``` yaml+jinja
# Create IPv6 DHCP Relay address
- panos_dhcp_relay:
    provider: '{{ provider }}'
    dhcp_interface: 'ethernet1/1'
    dhcp_relay_interface: 'ethernet1/1'
    ipv6_address: '2001:0db8:85a3:0000:0000:8a2e:0370:7334'
    interface: 'ethernet1/1'

# Delete DHCP Relay ipv6 address
- panos_dhcp_relay:
    provider: '{{ provider }}'
    dhcp_interface: 'ethernet1/1'
    dhcp_relay_interface: 'ethernet1/1'
    ipv6_address: '2001:0db8:85a3:0000:0000:8a2e:0370:7334'
    state: absent
```
