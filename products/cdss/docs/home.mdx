# Cloud-Delivered Security Services

Palo Alto Network's suite of Cloud-Delivered Security Services provide access to specialized subscription-based security solutions, designed specifically to defend against known, unknown, and advanced evasive threats. The threat data that is generated through advanced analysis is shared across the Palo Alto Networks security platforms to provide complete coverage across all threat vectors.

For more information about Palo Alto Networks CDSS offerings, refer to the [Palo Alto Networks product overview](https://www.paloaltonetworks.com/network-security/security-subscriptions).

For product documentation for CDSS products, refer the [Palo Alto Networks CDSS documentation portal](https://docs.paloaltonetworks.com/cdss).
