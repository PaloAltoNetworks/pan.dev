<MethodEndpoint method={"get"} path={"/api/sase/v1.0/resource/tenant/{tenant_id}/custom/{feature_name}/{request_name}"}></MethodEndpoint>



Returns all resource information, including available properties and details.
The following feature/resource name combinations are supported:

* mobileusers/mu_concurrent_connected_status

  Provides data on connected mobile users.

* locations/location_mobilegateways_status

  Provides data on mobile users service availability.
