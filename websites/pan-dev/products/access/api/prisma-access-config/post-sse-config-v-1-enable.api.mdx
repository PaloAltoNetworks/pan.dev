<MethodEndpoint method={"post"} path={"/sse/config/v1/enable"}></MethodEndpoint>



Set up application defaults for a newly on-boarded tenant. This replaces the initialization
performed by the Prisma Access Cloud-Managed user interface when a tenant is initially
created.

Use this API after you have [created a tenant](/sase/api/subscription/post-subscription-v-1-instances/) and claimed
your Prisma Access license for that tenant.
