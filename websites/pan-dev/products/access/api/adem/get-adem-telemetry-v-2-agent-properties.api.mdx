<h1 className={"openapi__heading"}>Get agent properties</h1>

<MethodEndpoint method={"get"} path={"/adem/telemetry/v2/agent/properties"}></MethodEndpoint>



Retrieves enpoint properties on which the Mobile User or Remote Networks agent is running.
These properties can include the device's
host name, public IP, geographic location, RAM and CPU usage using the UUID of the
agent installed on the Mobile User device. 

An agent_uuid or username filter is mandatory when using this
API. When used with Remote Networks a site_id must be specified.
