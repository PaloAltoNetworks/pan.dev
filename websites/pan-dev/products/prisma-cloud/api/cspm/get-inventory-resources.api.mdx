Lists resources. The response data depends on the table level you specify in the request body parameters.  

The **tableLevel** and **groupBy** request body parameters determine which 
response object fields a successful API request returns.

IF **tableLevel** is **1** and **groupBy** is unspecified, the response properties are as shown below:
```json
[
  {
    "cloudType": "",
    "serviceName": "",
    "totalResources": 0,
    "publicResources": ,
    "totalObjects": 0,
    "publicObjects": 0,
    "sensitiveObjects": 0,
    "malwareObjects": 0,
    "alerts": 0
  }
]
``` 
If **tableLevel** is **1** and **groupBy** is specified, the response properties are as shown below:
```json
[
  {
    "cloudType": "",
    "accountName": "",
    "regionName": "",
    "serviceName": "",
    "totalResources": 0,
    "publicResources": 0,
    "totalObjects": 0,
    "publicObjects": 0,
    "sensitiveObjects": 0,
    "malwareObjects": 0,
    "alerts": 0
  }
]
```

```json
[
```
