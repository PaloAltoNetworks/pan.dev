<h1 className={"openapi__heading"}>Get The Next Event Search Page</h1>

<MethodEndpoint method={"post"} path={"/search/event/page"}></MethodEndpoint>



Returns the next page of search results, using a token provided from the previous page. Used for when there are over 100 search results for a given RQL query.  

An initial request to perform an event search will return a **data.nextPageToken** that you 
can use as the initial **pageToken**.
