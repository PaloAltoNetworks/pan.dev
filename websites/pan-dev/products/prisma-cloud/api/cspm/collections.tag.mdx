Prisma Cloud Collections allow you to define logical groups of assets within Prisma Cloud which are meaningful to the stakeholders in your cloud environments. You can create Collections to represent applications and services, organize your cloud estate by business units, or maintain focused visibility on the security posture of critical components of your cloud infrastructure.
