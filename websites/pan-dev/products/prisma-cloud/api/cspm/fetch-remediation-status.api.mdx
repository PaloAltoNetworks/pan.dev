<h1 className={"openapi__heading"}>Get Remediation Status</h1>

<MethodEndpoint method={"post"} path={"/uve/api/v1/remediation/vuln-remediation-status"}></MethodEndpoint>



Get the remediation action status of assets. You can get the remediation status at two levels: 
- **Asset level** - Get the remediation status of an asset by asset ID
- **Group level** - Get the remediation status of all assets of a specific asset type. To get group level status, specify only the asset type and not the asset ID.

>**Note:** You need 'vulnerabilityRemediation' feature with 'View' permission to access this endpoint. Verify if your permission group includes this feature using the [Get Permission Group by ID](https://pan.dev/prisma-cloud/api/cspm/get-1/) endpoint. You can also check this in the Prisma Cloud console by ensuring that **Alerts > Remediate Vulnerabilities** is enabled.
