<MethodEndpoint method={"post"} path={"/license/api/v2/usage"}></MethodEndpoint>



Get a full breakdown of resources used per account for selected cloud types in a paginated way.

By default, you will receive your paginated usage data in the response object. You can control the number of items in the 
response object through the **limit** request body parameter. If data beyond what's in the response object is available,
the response object attribute **nextPageToken** will provide a token you can set in the request parameter **pageToken** of your
next request to get the next page of usage data.
