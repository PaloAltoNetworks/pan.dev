Generally, the data field in the response object contains the raw JSON blob as is received from the 
source cloud service provider API for the given resource.

Only the **rrn** parameter in the request body is used for this API. Ignore the **timelineItemId** and 
**findingType** fields for this API.

An example request body is:
```
{
  "rrn": "rrn::storageBucket:us-east-1:123456789012:test-bucket"
}
```
