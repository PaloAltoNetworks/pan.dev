<MethodEndpoint method={"get"} path={"/api/v1/tenant/{prismaId}/template/{id}"}></MethodEndpoint>



Returns the notification template with the specified ID. 

The caller must have one of the following Prisma Cloud roles:

* System Admin
* Account Group Admin
* Account and Cloud Provisioning Admin
* Account Group Read  Only
