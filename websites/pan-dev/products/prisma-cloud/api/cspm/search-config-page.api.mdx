<h1 className={"openapi__heading"}>Get The Next Config Search Page</h1>

<MethodEndpoint method={"post"} path={"/search/config/page"}></MethodEndpoint>



Returns the next page of search results, using a token provided from the previous page. Used for when there are over 100 search results for a given RQL query.

An initial config search request _([Perform Config Search by Query](https://pan.dev/prisma-cloud/api/cspm/search-config-by-query/),[Perform Config Search V2](https://pan.dev/prisma-cloud/api/cspm/search-config-v-2/))_ will return the `nextPageToken` field when more than 100 results are returned. The value of `nextPageToken` should be used for request parameter `pageToken` to retrieve the next page of results.
