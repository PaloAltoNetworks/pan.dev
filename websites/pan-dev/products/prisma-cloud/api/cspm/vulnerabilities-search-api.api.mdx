<h1 className={"openapi__heading"}>Get Vulnerabilities by RQL</h1>

<MethodEndpoint method={"post"} path={"/uve/api/v1/vulnerabilities/search"}></MethodEndpoint>



Get the list of vulnerabilities and their details based on an RQL query. For vulnerability RQL query attributes, see [Vulnerability Query Attributes](https://docs.prismacloud.io/en/enterprise-edition/content-collections/search-and-investigate/vulnerability-queries/vulnerability-query-attributes). For example queries, see [Vulnerability Query Examples](https://docs.prismacloud.io/en/enterprise-edition/content-collections/search-and-investigate/vulnerability-queries/vulnerability-query-examples).

 To download all the vulnerabilities by RQL in a CSV format, see [Download All Vulnerabilities by RQL](https://pan.dev/prisma-cloud/api/cspm/download-vulnerability-csv-file-in-investigate-table-view/) 
>**Note:** You need 'investigateVulnerabilityRql' feature with 'View' permission to access this endpoint. Verify if your permission group includes this feature using the [Get Permission Group by ID](https://pan.dev/prisma-cloud/api/cspm/get-1/) endpoint. You can also check this in the Prisma Cloud console by ensuring that **Investigate > Vulnerability** is enabled.
