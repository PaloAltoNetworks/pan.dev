<h1 className={"openapi__heading"}>Create Remediation Request</h1>

<MethodEndpoint method={"post"} path={"/uve/api/v1/remediation/vuln-create-remediation"}></MethodEndpoint>



You create one of the following remediation action request for an asset or a set of assets:
 - Create a task or JIRA ticket 
 - Create a merge request 
 - Suppress the vulnerability 

 Remediation action request can be created at the following levels:
- **Asset level**- Perform remediation action on an asset by asset ID
 - **Group level**- Perform remediation action for all the assets of a particular asset type for a specified CVE ID 
 - **Global level**- Perform remediation action on all assets of a specified CVE ID. 

>**Note:** You need 'vulnerabilityRemediation' feature with 'Create' permission to access this endpoint. Verify if your permission group includes this feature using the [Get Permission Group by ID](https://pan.dev/prisma-cloud/api/cspm/get-1/) endpoint. You can also check this in the Prisma Cloud console by ensuring that **Alerts > Remediate Vulnerabilities** is enabled.
