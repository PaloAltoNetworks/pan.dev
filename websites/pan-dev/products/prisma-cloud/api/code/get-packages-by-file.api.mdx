<h1 className={"openapi__heading"}>Get Packages Related to a File</h1>

<MethodEndpoint method={"get"} path={"/code/api/v1/vulnerabilities/packages/files/{fileMetadataId}"}></MethodEndpoint>



Gets all the Packages related to a specific file based on pagination
Pagination is required - this endpoint must use the pagination.validate
The API accepts a file metadata UUID.
There is an option to get the packages sorted by their error count.
root-only query parameter is optional - if true, returns only root level packages, in a sense of the file's dependency tree
