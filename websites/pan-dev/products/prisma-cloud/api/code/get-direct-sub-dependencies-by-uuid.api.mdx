<h1 className={"openapi__heading"}>Get Direct Sub-dependency Packages by Source Package</h1>

<MethodEndpoint method={"get"} path={"/code/api/v1/vulnerabilities/packages/{packageUuid}/direct-sub-dependencies"}></MethodEndpoint>



Gets all the direct sub-dependencies packages of a specific package.
The API accepts a package UUID.
There is an option to get the packages sorted by their error count.
