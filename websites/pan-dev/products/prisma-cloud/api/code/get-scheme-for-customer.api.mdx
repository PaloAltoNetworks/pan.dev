<MethodEndpoint method={"get"} path={"/code/api/v1/enforcement-rules"}></MethodEndpoint>



This API is used to fetch an array of all your existing Enforcement rules.
The response includes details of the rule and the behavior threshold configured to each Code category Prisma Cloud Application Security scans (IaC, Secrets, Images, Open source)
