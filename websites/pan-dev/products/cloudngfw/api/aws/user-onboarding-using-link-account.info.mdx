<h1 className={"openapi__heading"}>User Onboarding using Link Account</h1>



A link account is an unverified account used to onboard an AWS account to the Cloud NGFW service based on your input. A link account is required to run the Cloud Formation Template (CFT).  

>**Note:** You can add only one link account per subscription.
