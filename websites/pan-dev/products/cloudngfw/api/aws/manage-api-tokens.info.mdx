<h1 className={"openapi__heading"}>Manage API Tokens</h1>

For information about roles and permissions, see [Cloud NGFW roles and permissions](https://docs.paloaltonetworks.com/cloud-ngfw/aws/cloud-ngfw-on-aws/getting-started-with-cloud-ngfw-for-aws/invite-users). 
>**Note:** Cloud NGFW Programmatic access is disabled by default.
