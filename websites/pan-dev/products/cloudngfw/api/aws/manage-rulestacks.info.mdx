<h1 className={"openapi__heading"}>Manage Rulestacks</h1>



Rulestacks define access control (App-ID, URL Filtering) and threat prevention behavior for the Cloud NGFW resources using security rules, associated objects, and profiles. The Cloud NGFW resources use the rulestack definitions to protect the traffic uisng a two-step process. First, it enforces rules to allow or deny your traffic. Then, it inspects content on the allowed traffic based on the Security Profiles you create.
