# Cloud NGFW Overview

Cloud NGFW for AWS is Palo Alto Networks ML-powered Next-Generation Firewall (NGFW) capability delivered as a fully managed cloud-native service by Palo Alto Networks on the Amazon Web Services (AWS) platform. 

This deployment model combines the power of the Palo Alto NGFW with the ease of use.
