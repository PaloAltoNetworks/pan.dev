# IKE Gateway Management

<ApiLogo logo={{"url":"pan-logo.png"}} darkLogo={undefined}></ApiLogo>

Manages Internet Key Exchange (IKE) Gateway network profiles, which specify
the configuration information necessary to perform IKE protocol negotiation with a
peer gateway. This is the Phase 1 portion of the IKE/IPSec VPN setup.
