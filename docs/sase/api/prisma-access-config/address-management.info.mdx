# Address Management

<ApiLogo logo={{"url":"pan-logo.png"}} darkLogo={undefined}></ApiLogo>

Manages [address objects](https://docs.paloaltonetworks.com/network-security/security-policy/objects/addresses). These allow you to reuse the same object as a source or
destination address across all the policy rulebases without having to add the address
manually each time.
