## List RuleStack



Retrieve the metadata of the specified rulestack.  

You can also query rulestacks based on the tags using this command, `/v1/config/rulestacks?tags=&lt;string&gt;`.   
This command lists the rulestacks where the tag name starts with the string.  

&gt;DynamoDB supports this command, and should be implemented as a library.
